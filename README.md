# TabSeonuk

A minimal Chrome extension that replaces the new tab page with [seon.uk](https://seon.uk), while keeping the address bar empty and focused — just like a native new tab.

## Install

### From Chrome Web Store
*(Coming soon)*

### Manual
1. Clone or download this repo
2. Go to `chrome://extensions`
3. Enable **Developer mode**
4. Click **Load unpacked** and select this folder

## How it works

Overrides Chrome's new tab page with an iframe pointing to `seon.uk`. The address bar stays clean and ready to type.
